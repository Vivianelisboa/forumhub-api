# forumhub-api
ForumHub API

API REST de um fórum desenvolvida com Java e Spring Boot.

Tecnologias utilizadas

Java 17

Spring Boot

Spring Data JPA

Spring Security

MySQL

Flyway

Maven

Funcionalidades

Criar tópico

Listar tópicos

Buscar tópico por ID

Atualizar tópico

Excluir tópico

Autenticação de usuário

Como executar o projeto
1. Criar o banco de dados

Execute no MySQL:

CREATE DATABASE forumhub;
2. Rodar a aplicação

No terminal, dentro da pasta do projeto:

mvn spring-boot:run
Acesso à API

Após iniciar o projeto:

http://localhost:8080
Documentação da API (Swagger)
http://localhost:8080/swagger-ui.html
